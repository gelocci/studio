\# Gelocci Studio



O Gelocci Studio é uma plataforma interna de agentes criada para apoiar a evolução contínua do ecossistema Gelocci, especialmente o site gelocci.com.br.



Ele funciona como uma equipe digital de apoio ao Gerson, auxiliando em arquitetura, desenvolvimento, revisão, testes, segurança, publicação, SEO, produto, conteúdo e evolução das ferramentas financeiras.



\## Objetivo



Apoiar a criação, evolução e manutenção dos produtos digitais da família Gelocci, mantendo o gelocci.com.br leve, rápido, organizado e orientado a valor para o usuário.



\## Escopo inicial



\- Documentar o conceito do Gelocci Studio.

\- Definir os agentes e suas responsabilidades.

\- Criar o system design da plataforma.

\- Definir fluxos de decisão e aprovação.

\- Criar os primeiros protótipos.

\- Evoluir para uma ferramenta interna de análise e apoio ao desenvolvimento.



\## Premissas



\- O Gelocci Studio é uma ferramenta interna, não um produto comercial.

\- O Gerson mantém a decisão final sobre qualquer alteração.

\- O gelocci.com.br deve permanecer leve, preferencialmente em HTML5, CSS e JavaScript puro.

\- Backend só deve ser usado quando houver justificativa real.

\- Toda recomendação relevante deve ser revisada antes de virar implementação.
